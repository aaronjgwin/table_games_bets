# Table Games Bets

A comprehensive toolset for analyzing and managing betting strategies for tabletop and casino-style games. This repository provides data structures and logic to simulate game outcomes and calculate betting odds.

## 🚀 Features

* **Game Simulations**: Built-in logic for various table games to test betting systems.
* **Odds Calculation**: Automated tools for determining house edge and player win probabilities.
* **Strategy Testing**: Framework for implementing and stress-testing custom betting progression systems (e.g., Martingale, Fibonacci).
* **Data Visualization**: (If applicable) Visual reports on bankroll volatility and long-term expected value.
